Video

Day 1 in-person in Austin, created a TikTok clone with scrolling video playback. After the PoC, I wanted to learn more about streaming videos. This led to creating a HLS video processing service, which I shared for fellow developers to use. Using FFMPEG, it creates multiple resolutions of an mp4 and breaks them into chunks to be streamed to the client. It also creates a thumbnail for the video.

I changed the project to Gauntlet Exchange for users to exchange information. The idea is record a teaching video, then upload and analyze it. The video is transcribed and analyzed for filler words with a report generated to help the user improve.

Technologies

  • FFMPEG for video processing
  • Azure Queue Storage for video processing queues
  • Kubernetes for microservices that auto-scale to meet queue and compute demands
  • Azure Speech Services for transcribing the video
  • OpenAI for analyzing the transcript
  • WebView for Android & iOS mobile apps

Older AI Projects/Startups

Vortext (I Am)

2018

During a TechStars Startup Weekend, I led a team of three to design and develop a messaging hub that unified email, social media, and text into one app. Using Azure Machine Learning and Xamarin, we prioritized urgent messages with an Eisenhower matrix model, simplifying workflows and reducing the need to check multiple apps. Over 54 hours, we rapidly iterated on feedback from judges and mentors, delivering a proof-of-concept that secured 1st place. Post-event, I collaborated with investors, mentors, and early users to refine features, optimize messaging flow, and grow adoption.

Technologies

  • Natural Language Processing (NLP) pipeline with intent recognition

Dioth (Raz Recognition)

2014

What started as a summer project to help a quadriplegic friend play World of Warcraft independently turned into a larger effort to improve gaming accessibility. I designed and developed the desktop application and API using C#, ASP.NET Web API, and Azure technologies while gathering feedback from over 3,000 handicapped gamers. The project gained momentum when it was selected at the University of Houston's 3-Day Startup event, where I led a team of six to iterate on the idea. We pitched it and won Microsoft BizSpark, securing $120,000 in Azure credits.

Technologies

  • Windows Speech Recognition (STT)
